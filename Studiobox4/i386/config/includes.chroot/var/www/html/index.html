Installer Airtime

Avant de pouvoir diffuser une emission de radio, vous devez installer et configurer le logiciel Airtime. Vous devez pour cela avoir au préalable initié une connexion à internet.

Airtime est préinstallé sur le système Studiobox 3.0 et sa mise en place peut s’effectuer simplement en procédant aux opérations suivantes :

1- Ouvrir un terminal (clic-droit sur le bureau -> Utilitaires -> Terminal)

2- Entrer la commande suivante en respectant bien la syntaxe :

sudo ./install-airtime

3-Une fois le script terminé, relancer«Programmer une diffusion radio» dans le menu.

4- Valider les différentes étapes de la configuration sans apporter de changements. A la cinquième étape, veillez à bien entrer dans un terminal les commandes qui vous sont proposées. Vous pouvez proceder à un copier/coller dans le terminal des commandes qui s'affichent dans le navigateur.

5- En fin de processus de configuration, identifiez vous à Airtime avec l’identfiant «admin» et le mot de passe «admin».

Choisissezla langue de l’interface.

Configurez ensuite définitivement votre logiciel en allant dans le menu «Système»-> «Préférences». Notez que le réglage du fuseau horaire s'effectue à partir du lien "admin" dans la barre de menu.
